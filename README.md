# 5. Main Function 
* 회원가입 / 로그인 
* 나의 기본 정보 / 프로필 사진 변경 
* 나의 이야기 남기기 
* 누군가의 이야기 랜덤으로 읽기
* 고해성사 서비스 
* 익명 채팅방 개설 및 채팅 기능 
* "따봉" 가장 많이 받은 이야기 피드 서비스 (weekly)
